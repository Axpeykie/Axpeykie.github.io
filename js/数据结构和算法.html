# 选择排序

一个长度为n的数组，每遍历一次这个数组选出一个最小或最大的值，把这个值加入到新数组中，对剩下的n-1个数再进行遍历，直到这个数组的长度为0，最终得到一个有序排列的新数组。

时间复杂度：O(n2)

# 快速排序

选取一个基准值，把小于基准值的数放到一边，大于基准值的数放到另一边，对两边的数组进行同样的操作，最后得到一个有序数组。思想：分而治之；方法：递归。

时间复杂度：平均(最佳)情况为O(nlogn) 最糟情况为O(n2) 可以通过随机选择基准值来使每次输入都能得到较好的效果。

空间复杂度：O(logn) 即递归调用的栈高度

# 冒泡排序

遍历数组每一个值，把前一个值和后一个值比较，如果前一个值比后一个值大就将他们俩换位。遍历一次数组后数组末尾的值会是最大的值。再对前面的n-1个元素进行相同的操作，每次遍历将最大的值移到最后。
